# ElectionResultsPredictor
This Python project predicts the winner in elections based on social media and poll data.

This project is an AI project. This includes ML models usage and a simple DL model.
By uploading csv files(poll data and social media data), we can be able to see the dashboard that consists the data analysis and the final winner of elections at the end.

To get the csv files that has the correct format as included in code, check the data folder in this exact same repository.

Upon running the cells, the user gets to see a streamlit link at one cell. By clicking on the link the actual results are displayed.

There are 2 cells at the end that are written to get random csv files with the same format that is needed by the code.
